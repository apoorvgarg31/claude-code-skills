# Contributing to dev-workflow

Thanks for your interest in contributing!

## How to Contribute

1. Fork the repo
2. Create a branch (`git checkout -b feature/my-feature`)
3. Make your changes
4. Commit with descriptive messages
5. Push and open a PR

## Development

```bash
# Clone
git clone https://github.com/apoorvgarg31/claude-code-skills.git
cd claude-code-skills/skills/dev-workflow

# Test scripts
./scripts/init-workflow.sh test-project
./scripts/workflow-status.sh
```

## Adding New Sub-Agents

1. Create `agents/your-agent.md`
2. Follow existing agent format
3. Add to SKILL.md workflow
4. Update README

## Code Style

- Clear, descriptive names
- Comments for complex logic
- Follow existing patterns
